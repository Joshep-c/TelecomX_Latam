# 📊 Telecom X — Análisis de Evasión de Clientes (Churn)

Análisis exploratorio de datos (EDA) para identificar los factores asociados a la cancelación de clientes en Telecom X.

## 🛠️ Tecnologías
Python, Pandas, NumPy, Matplotlib, Seaborn

## 📁 Estructura
- **Extracción**: carga de datos desde API en formato JSON
- **Transformación**: limpieza, normalización y estandarización
- **Análisis**: visualizaciones y métricas para identificar patrones de evasión

## 🔍 Principales Hallazgos
- El **26.5%** de los clientes cancelaron el servicio
- Los clientes con contrato **mes a mes** presentan la mayor tasa de abandono
- La mayoría de las cancelaciones ocurren en los **primeros meses** de contrato
- Los clientes con **fibra óptica** y pago por **cheque electrónico** tienen mayor tendencia a cancelar
